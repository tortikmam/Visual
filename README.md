# Visual
Клишин Владимир Евгеньевич ИКС-431

Этот репозиторий сделан для лаюораторных работ по Визуальному программированию<br>
В проекте есть несколько Activity:<br>
    GpsActivity - получает координаты по gps<br>
    KukulatorActivity - обычный калькулятор :><br>
    MainActivity - главное activity для перехода на другие окна<br>
    MediaPlayerActivity - мущыкальный плеер<br>
    MobileNetActivity - получает информацию о сети пользователя<br>
    SocketsActivity - передает сообщения с клиента на сервер<br>
