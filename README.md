# Visual
Клишин Владимир Евгеньевич ИКС-431

Этот репозиторий сделан для лаюораторных работ по Визуальному программированию
В проекте есть несколько Activity:
    GpsActivity - получает координаты по gps
    KukulatorActivity - обычный калькулятор :>
    MainActivity - главное activity для перехода на другие окна
    MediaPlayerActivity - мущыкальный плеер
    MobileNetActivity - получает информацию о сети пользователя
    SocketsActivity - передает сообщения с клиента на сервер
